Processor en geheugen

Algemeen

De processor is een onmisbaar onderdeel in een computer. De processor bepaalt namelijk voor een heel groot deel de prestaties van je computer. En daarbij heb je hem ook echt nodig, want zonder processor zal een computer niet werken (niet eens opstarten). Processor afgekort is CPU (Central Processing Unit).

Het hart van de computer

De processor wordt ook wel het hart van de computer genoemd, want de processor is namelijk verantwoordelijk voor alle berekeningen en aansturingen die in een computer plaatsvinden. Oftewel, hoe sneller de processor, hoe meer berekeningen en aansturingen er per seconde kunnen worden verwerkt.

De snelheid van een processor

Er zijn verschillende factoren die de snelheid van een processor bepalen:

Aantal processorkernen; Vroeger was een processor gewoon één processor. Tegenwoordig kunnen er meerdere processorkernen in één processor zitten. Als er 2 processorkernen in een processor zitten dan noemen we dit een Dual Core processor. Bij 4 processorkernen in een processor noemen we dit een Quad Core processor. En bij 6 processorkernen noemen we dit een Hexa Core processor.

De kloksnelheid;De kloksnelheid zegt eigenlijk hoeveel taken een processorkern in 1 seconde kan doen. Dus hoe hoger de snelheid, hoe meer taken een processor per seconde kan doen. De kloksnelheid wordt uitgedrukt in Herz (Hz), maar tegenwoordig worden processoren zo snel dat er tegenwoordig in Giga Herz (GHz) wordt aangegeven wat de kloksnelheid van een processor is.

Het cachegeheugen; Om berekeningen te kunnen doen, moet de processor gegevens hebben om aan de slag te kunnen gaan en deze moet de processor ergens vandaan halen. Om dit zo snel mogelijk te doen is er een stukje geheugen in de processor geplaatst. Dit heet het cachegeheugen. Dus hoe groter het cachegeheugen, hoe meer gegevens de processor snel bij de hand heeft.
